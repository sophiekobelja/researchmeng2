Derby Public API

Derby's public API consists of two sets of javadoc:

  • JDBC2/JDBC3 Public API - Consult this javadoc if your application runs on J2SE 1.3, J2SE 1.4, or J2SE 5.0. Also consult this javadoc if your application runs with the CDC/Foundation Profile(JSR-169).
  • JDBC4 Public API - Consult this javadoc if your application runs on Java SE 6.
